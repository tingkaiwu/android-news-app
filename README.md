## Introduction

Tinder-Like News App is a Tinder flavor news app based on Google Component Architectural MVVM Pattern. The purpose of this project was to learn how to build a Android app. In addition, I also implemented the bottom bar & page navigation using JetPack navigation component, and utilized Mindorksâ€™ PlaceHolderView to support swipe gestures for liking/disliking the news. I	Built the Room Database with LiveData & ViewModel to support local cache and offline model.	Integrated Retrofit and Rxjava to pull the latest news data backed by API from newsapi.org

## Requirement

- Android Studio

git clone git@github.com:tingkaiwu/android-news-app.git
